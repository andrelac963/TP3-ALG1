# TP3-ALG1
Trabalho prático 3 de Algoritmos 1

## Descrição

Este projeto resolve o problema da rainha e dos peões em um tabuleiro personalizado, utilizando uma solução exata e uma solução aproximada para comparar eficiência e qualidade.

## Como Executar

### Compilação

Compile o programa com:

```
make
```

Para limpar os arquivos compilados:

```
make clean
```

### Execução

#### Somente Solução Exata

```
./tp3 < test/instancia_X.txt
```

#### Solução Exata e Aproximada (Modo Verbose)

```
./tp3 -v < test/instancia_X.txt
```
